# DatasetNomfundo-
Dataset for Signal Detection / Spectrum Sensing
This dataset is a MATLAB .mat file designed for signal detection and spectrum sensing research in wireless communication systems. It contains labeled samples representing the presence and absence of a signal under realistic channel conditions modeled using the 3GPP CDL-A channel model.

The dataset is suitable for applications in:

Cognitive radio systems
Spectrum sensing algorithms
Machine learning-based signal detection
Wireless communication research and simulations
